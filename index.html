Ojs-base

Components for OCaml/Javascript applications.

This is work in progress, interfaces will change. It is currently used by the preview server of Stog (>0.14.0), so a 0.1.0 release was mandatory.

Download

Ojs-base is hosted on Github.

Releases
  • 0.1.0 [2014/10/21]: initial release.
Installation

To be installed, Ojs-base requires:

  • OCaml >= 4.02.0,
  • Js_of_ocaml >= 2.4.0,
  • Lwt >= 2.4.6,
  • Cohttp > 0.11.2.

To compile and install:

  • tar xvfz ojs-base-XXX.tar.gz
  • cd ojs-base-XXX
  • make all install

This will install the ojs* packages with findlib.

You can also install with Opam:

opam install ojs-base
