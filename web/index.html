To be installed, Ojs-base requires:

  • OCaml >= 4.02.1,
  • Js_of_ocaml >= 2.5,
  • Lwt >= 2.4.8,
  • Cohttp >= 0.18.2,
  • Websocket > 2.1,
  • Yojson >= 1.1.8,
  • Ppx-deriving-yojson >= 2.3.
  • Base64 >= 2.0.
  • Xtmpl >= 0.13.0.
  • Magic-mime >= 1.0.

To compile and install:

  • tar xvfz ojs-base-XXX.tar.gz
  • cd ojs-base-XXX
  • make all install

This will install the ojs* packages with findlib.

You can also install with Opam:

opam install ojs-base

Reference documentation is here.
